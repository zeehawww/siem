How It Works

Choose an attack technique, set a simulated target, and run the simulation. The system generates synthetic events matching that technique and runs them through the detection engine in simulation mode — without affecting live state or persisted alerts. You'll see exactly which rules fire, the alert types generated, and WHY each alert was triggered (explainability). Think of it as a built-in Breach & Attack Simulation (BAS).
