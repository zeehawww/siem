What is Alert DNA?

Traditional SIEMs correlate alerts by entity (same IP, same user). Alert DNA fingerprinting correlates alerts by behavioral pattern: time-of-day bucket, event type, MITRE tactic, severity, and origin (internal vs external). Two unrelated attackers using the same technique share the same DNA — enabling cross-entity pattern detection impossible in standard SIEMs.
