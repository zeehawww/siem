This mini SIEM focuses on a few common SSH‑related behaviors:

  • BRUTE_FORCE – 3+ consecutive failed logins from the same IP.
  • COMPROMISED_ACCOUNT – success after multiple failures.
  • PRIV_ESC – sudo usage events.
  • ROOT_LOGIN – direct root login detection.
