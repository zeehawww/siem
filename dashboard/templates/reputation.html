How Reputation Scoring Works
Score Accumulation
CRITICAL alert → +50 pts, HIGH → +30 pts, MEDIUM → +15 pts, LOW → +5 pts. Type bonuses: ROOT_LOGIN adds +30, COMPROMISED_ACCOUNT adds +25.
Behavioral Decay
Every pipeline run, all entities that were NOT seen this run have their score multiplied by 0.85. An entity that goes quiet for 5 runs loses ~44% of their score.
vs. Commercial SIEMs
Splunk, QRadar, and Microsoft Sentinel rely on paid threat intel (VirusTotal, Shodan). This system builds reputation organically from your own telemetry — no API keys needed.
